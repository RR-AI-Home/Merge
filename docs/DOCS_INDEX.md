## Current Structure

- `ROLES/` - agent role guides.
- `sprints/active/` - sprint docs that drive current work.
- `sprints/completed/` - implemented or closed sprint records.
- `sprints/canceled/` - superseded sprint records preserved as source/reference material only.
- `qa/` - detailed QA reports when a run needs more than a [QA_RUNS.md](./QA_RUNS.md) entry.
- `security/` - security reports and appsec notes.
- `mockups/` - local browser mockups and visual proofs.
- `superpowers/` - design specs and implementation plans created by Superpowers workflows.
- `appone-reference/` - copied AppOne reference material; do not treat as active Merge instructions unless deliberately converted.
